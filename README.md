# Tasks API (Go, net/http)

##Practical assignments are implemented in the `master` branch

Minimal backend service for managing tasks (todos).

## Demo Video
🎬 https://drive.google.com/file/d/1uA5E72XHza7ywmypQNvVCVwEzJxmMbMk/view?usp=sharing

## Features
- GET /tasks
- GET /tasks?id=1
- POST /tasks
- PATCH /tasks?id=1
- API key middleware
- Logging middleware
- In-memory storage
